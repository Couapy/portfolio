mc-manager

Web manager de serveurs Minecraft

Description

mc-manager est un web manager permettant d'administrer des serveurs Minecraft.

Ce projet répond à une problématique rencontrée avec des amis lors de partie de jeu, où les serveurs sont executés sur un serveur personnel. Pour réduire les coûts d'électricité, les serveurs Minecraft ne sont pas allumés en permanence, et je devais les allumer et les éteindre en me connectant en SSH à chaque fois. Cependant je n'étais pas toujours disponible pour lancer les serveurs lorsque mes amis voulaient continuer la partie.

Ainsi l'idée m'est venue de créer un web manager pour qu'ils puissent tout faire sans mon aide.

Le site permet alors de créer un serveur Minecraft, de choisir sa version et de le configurer. De plus, ne voulant pas perdre l'avantage du jeu multijoueur, j'ai décidé d'ajouter un système de partages permettant aux joueurs du même serveur de démarrer ou d'arrêter un serveur Minecraft.

Le projet est-il fini ?
Je ne pense pas, je songe à ajouter quelques fonctionnalités comme :

  • Accès à la console depuis le navigateur
  • Gestion des maps (upload / suppression / sélection)
  • Ouverture automatisée des ports dans la livebox
